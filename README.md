Rock Paper Scissors Game
This is a simple Rock Paper Scissors game built using HTML, CSS, and JavaScript. The game allows a user to play against a computer, tracks wins, losses, and ties, and includes an Auto Play mode that automatically plays the game every second.

Features
Interactive UI with styled buttons and icons

Game logic to determine win, lose, or tie

Persistent score tracking using localStorage

Auto Play mode to simulate games automatically

Reset button to clear the current score

Technologies Used
HTML

CSS

JavaScript

localStorage (for saving scores)
